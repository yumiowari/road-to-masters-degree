## Árvore Binária de Busca

Implementação de uma estrutura de dados de Árvore Dinâmica (baseada em hierarquia) com critério de inserção por busca binária, caracterizando uma Árvore Binária de Busca (BST - _Binary Search Tree_).

### Funções...

- __plant()__ aloca a memória da árvore binária;

- __getsize()__ acessa a quantidade de elementos na árvore;

- __branch()__ insere um elemento na árvore;

- __disbranch()__ remove um elemento da árvore;

- __cutdown()__ libera a memória da árvore;

- __showtree()__ percorre a árvore:

    - __inorder()__ em ordem;

    - __preorder()__ pré-ordem; e

    - __postorder()__ pós-ordem.

- __find()__ busca o elemento na árvore.
